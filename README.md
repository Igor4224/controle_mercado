# controle_mercado

Desenvolvi um sistema de gestão comercial utilizando Microsoft Excel com foco no controle operacional e acompanhamento dos resultados de um mercado varejista.
O projeto foi estruturado para centralizar informações de estoque, vendas, entregas e indicadores financeiros, proporcionando maior controle das operações e suporte à tomada de decisões.

Principais funcionalidades:
• Controle automatizado de estoque com atualização dinâmica das quantidades;
• Monitoramento de produtos com necessidade de reposição;
• Registro e acompanhamento de vendas semanais;
• Controle de entregas por região;
• Cálculo automático de receitas e resultados financeiros;
• Consolidação de indicadores operacionais e gerenciais;
• Relatórios analíticos para acompanhamento do desempenho do negócio.

 Recursos utilizados:
• Microsoft Excel Avançado;
• Fórmulas estruturadas;
• PROCV (VLOOKUP);
• SOMASE (SUMIF);
• ÍNDICE e CORRESP;
• SE e SEERRO;
• Tabelas estruturadas;
• Relatórios e análises gerenciais.
